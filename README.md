## SclavGEM: Genome-scale metabolic model for Streptomyces clavuligerus

#### Description

This repository contains the latest Streptomyces clavuligerus model, based on iDG1237


#### Citation

The model iDG1237 was downloaded from the supplementary data from the following publication:

Gómez-Ríos, D., López-Agudelo, V. A., Ramírez-Malule, H., Neubauer, P., Junne, S., Ochoa, S., & Ríos-Estepa, R. (2020). A genome-scale insight into the effect of shear stress during the fed-batch production of clavulanic acid by Streptomyces clavuligerus. Microorganisms, 8(9), 1255. doi:10.3390/microorganisms8091255
