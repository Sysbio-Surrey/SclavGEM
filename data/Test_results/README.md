# Description of the tests

The following tests were performed:

## Memote
https://github.com/opencobra/memote
Memote version 0.17.0 was installed according to installation instruction.

The following command was used to create the report

`memote report snapshot --filename "memote_report.html" path/to/model.xml`

## Macaw
https://github.com/Devlin-Moyer/macaw
Macaw version 1.0.0 was installed according to the instructions.
Macaw results were generated by the script in:
code/generate_output_model_tests/run_macaw.py

## FROG
FROG results were generated via an online webserver:
https://www.ebi.ac.uk/biomodels/curation/fbc
